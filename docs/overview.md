ASI / GSI — Document Overview
This folder contains the core materials describing the architecture, semantics, and interaction model of the ASI/GSI dual‑loop innovation protocol.
It provides a structured entry point into the two foundational books and the accompanying specifications.

1. ASI — Semantic Infrastructure for Ideas
ASI_EN.pdf  
A full conceptual and architectural description of ASI — a semantic system that evaluates, structures, and routes ideas.

Key themes:

Semantic evaluation of ideas

Novelty detection and conceptual mapping

Routing ideas to relevant organizational contexts

Talent–idea alignment

The role of ASI as the “idea loop” in the dual‑loop architecture

2. GSI — Global Interaction Protocol
GSI_EN.pdf  
A full description of GSI — a global protocol that connects problems with those capable of solving them.

Key themes:

Request/response architecture between organizations

Problem–solution routing

Federated interaction model

Organizational interoperability

The “interaction loop” complementing ASI

3. Dual‑Loop Architecture (ASI + GSI)
Together, ASI and GSI form a unified innovation infrastructure:

ASI processes ideas

GSI processes interactions

Combined, they create a global network where ideas and problems flow toward the entities capable of addressing them.

This dual‑loop model transforms fragmented innovation ecosystems into a connected global system.

4. Specifications (RFC‑style)
The repository also includes (or will include) formal specifications:

ASI RFC — Semantic Protocol Specification

GSI RFC — Interaction Protocol Specification

These documents define the protocol behavior, message formats, routing logic, and interoperability rules.
